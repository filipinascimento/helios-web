# Basic Example

This is the smallest browser example in the repository. It creates an
eight-node graph, starts a `Helios` renderer, and leaves UI panels disabled so
the core package contract stays easy to inspect.

Run the Vite dev server and open:

```bash
http://localhost:5173/docs/examples/basic/
```

The full application used by the dev server root lives in [`docs/app`](../../app/).
